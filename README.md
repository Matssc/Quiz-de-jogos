Este projeto foi desenvolvido como parte de um Projeto Integrador da faculdade, com o objetivo de aplicar conceitos de:
Desenvolvimento em Java
operações CRUD
integração com Banco de Dados
trabalho em equipe
o sistema consiste em um Quiz de Games, onde é possível responder perguntas e também realizar o gerenciamento completo das questões através de um CRUD integrado ao banco de dados.
